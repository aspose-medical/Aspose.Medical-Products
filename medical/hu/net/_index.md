<p>Az <strong>Aspose.Medical for .NET</strong> egy erőteljes, tiszta .NET osztálykönyvtár DICOM (Digital Imaging and Communications in Medicine) fájlokkal való munkához. Lehetővé teszi a fejlesztők számára az orvosi képalkotási adatok programozott olvasását, írását, módosítását, anonimizálását, konvertálását és átkódolását külső szoftver vagy natív függőségek szükségessége nélkül.</p>

<p>A könyvtár teljes egészében felügyelt C# kódban íródott, biztosítva a valódi többplatformos kompatibilitást Windows, Linux és macOS rendszereken. Natív binárisok vagy harmadik fél függőségek nélkül az Aspose.Medical zökkenőmentesen integrálódik bármely .NET alkalmazásba, az asztali eszközöktől a felhőalapú egészségügyi rendszerekig.</p>

<p><strong>Főbb jellemzők:</strong>
<ul>
    <li>DICOM fájlok olvasása és írása az összes szabványos DICOM adatelem és értékreprezentáció teljes támogatásával.</li>
    <li>DICOM fájlok anonimizálása a betegek magánéletének védelme és a HIPAA megfelelőség biztosítása érdekében konfigurálható bizalmasság profilok használatával.</li>
    <li>DICOM képek konvertálása általános formátumokba: JPEG, PNG, TIFF és vissza DICOM-ba raszterképekből.</li>
    <li>DICOM adatok szerializálása és deszerializálása JSON-ba és XML-be modern webszolgáltatásokkal és API-kkal való integrációhoz.</li>
    <li>Átkódolás átviteli szintaxisok között, beleértve a JPEG, JPEG 2000, HTJ2K, JPEG-LS és RLE tömörítést.</li>
    <li>Teljes támogatás többkockás DICOM képekhez és képátfedésekhez.</li>
    <li>Fejlett képfeldolgozás modalitás és VOI keresési táblákkal a megfelelő ablak és szint beállításához.</li>
    <li>HTJ2K (High-Throughput JPEG 2000) támogatás az új generációs orvosi képalkotási tömörítéshez.</li>
    <li><strong>Hamarosan elérhető:</strong> DICOM NIfTI konverzió neuroimaging és kutatási alkalmazásokhoz.</li>
</ul>
</p>

<p>Az Aspose.Medical for .NET egy önálló, önellátó könyvtár, amely nem igényel semmilyen DICOM megtekintő, orvosi képalkotó szoftver vagy natív kodekek telepítését. Ideális egészségügyi informatikai rendszerekhez, PACS integrációhoz, telemedicina platformokhoz és bármely megbízható DICOM feldolgozást igénylő alkalmazáshoz.</p>

    <h2 class="h2title">
     Tiszta .NET könyvtár - nulla függőség
    </h2>

Ellentétben más orvosi képalkotó könyvtárakkal, amelyek natív kodekekre vagy külső eszközökre támaszkodnak, az Aspose.Medical teljesen önálló. Minden képtömörítési kodek (JPEG, JPEG 2000, HTJ2K, JPEG-LS, RLE) tiszta .NET-ben van implementálva, biztosítva a zökkenőmentes telepítést Windows, Linux és macOS rendszereken platform-specifikus konfiguráció nélkül. Ez ideálissá teszi konténeres telepítésekhez, felhőkörnyezetekhez és olyan forgatókönyvekhez, ahol natív könyvtárak telepítése nem megvalósítható.

   <div class="col-lg-12">
    <h2 class="h2title">
     DICOM fájlok anonimizálása adatvédelmi megfelelőséghez
    </h2>
    <p>
     A betegek magánéletének védelme kritikus fontosságú az egészségügyi alkalmazásokban. Az Aspose.Medical for .NET átfogó DICOM anonimizálási képességeket biztosít a DICOM PS 3.15 bizalmasság profilok alapján. Eltávolíthatja vagy módosíthatja a beteg azonosító információkat (PII) miközben megőrzi a képalkotási adatok klinikai értékét. A könyvtár támogat konfigurálható anonimizálási profilokat, beleértve opciókat a biztonságos privát attribútumok, UID-k, eszköz és intézményi azonosítók, beteg jellemzők és longitudinális időbeli információk megtartására.
    </p>
    <div class="codeblock" id="code">
     <h3>
      DICOM fájl anonimizálása - C#
     </h3>
     <pre><code class="cs">// DICOM fájl betöltése
using var dicomFile = DicomFile.Open("patient_scan.dcm");

// Anonimizáló létrehozása alapértelmezett profillal
var profile = ConfidentialityProfile.CreateDefault();
var anonymizer = new Anonymizer(profile);

// Anonimizálás és mentés
anonymizer.Anonymize(dicomFile, "anonymized_scan.dcm");</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     DICOM konvertálása képformátumokba .NET C#-ban
    </h2>
    <p>
     Az Aspose.Medical for .NET megkönnyíti a DICOM képek konvertálását szabványos képformátumokba megtekintéshez, megosztáshoz vagy nem orvosi rendszerekkel való integrációhoz. A könyvtár kezeli a DICOM képrenderelés teljes összetettségét, beleértve a modalitás LUT-ok, VOI LUT-ok (ablak/szint) és fotometrikus értelmezés megfelelő alkalmazását. Akár miniatűröket kell generálnia, képexportokat létrehoznia vagy DICOM megtekintőt építenie, az API minimális kóddal magas minőségű renderelést biztosít.
    </p>
    <div class="codeblock" id="code">
     <h3>
      DICOM konvertálása PNG-be - C#
     </h3>
     <pre><code class="cs">// DICOM fájl betöltése
using var dicomFile = DicomFile.Open("chest_xray.dcm");

// Renderelés képpé
var renderOptions = new GrayscaleRenderOptions
{
    WindowWidth = 400,
    WindowCenter = 40
};
var image = dicomFile.RenderImage(renderOptions);

// Mentés PNG-ként
image.Save("chest_xray.png");</code></pre>
    </div>

    <h2 class="h2title">
     Átkódolás átviteli szintaxisok között
    </h2>
    <p>
     A különböző orvosi képalkotó rendszerek és archívumok különböző DICOM átviteli szintaxisokat igényelhetnek. Az Aspose.Medical for .NET lehetővé teszi az átkódolást különböző átviteli szintaxisok között, beleértve a nem tömörített formátumokat, JPEG Baseline-t, JPEG Lossless-t, JPEG 2000-t, HTJ2K-t (High-Throughput JPEG 2000), JPEG-LS-t és RLE-t. Ez elengedhetetlen a PACS integrációhoz, archívum optimalizálásához és a kompatibilitás biztosításához különböző egészségügyi rendszerek között. Minden kodek tiszta .NET-ben van implementálva, garantálva a konzisztens viselkedést az összes platformon.
    </p>

     DICOM JSON és XML szerializáció
    </h2>
    <p>

Az Aspose.Medical for .NET teljes támogatást nyújt a DICOM adatok JSON-ba (a DICOM JSON modell szerint) és XML formátumokba való szerializálásához. Ez lehetővé teszi a zökkenőmentes integrációt RESTful API-kkal, FHIR alapú rendszerekkel és webes alkalmazásokkal. A szerializáció kétirányú - JSON-t és XML-t is visszaelemezhet DICOM adathalmazokká.

A NIfTI az agyi képalkotás kutatásának szabványos formátuma, és a hamarosan elérhető képesség a klinikai DICOM adatok NIfTI-vé való konvertálására lehetőségeket nyit a fejlett elemzéshez, gépi tanuláshoz és kutatási munkafolyamatokhoz.
