<p><strong>Aspose.Medical for .NET</strong> je výkonná, čistá třídní knihovna .NET pro práci se soubory DICOM (Digital Imaging and Communications in Medicine). Umožňuje vývojářům programově číst, zapisovat, upravovat, anonymizovat, konvertovat a překódovávat data lékařského zobrazování bez nutnosti jakéhokoli externího softwaru nebo nativních závislostí.</p>

<p>Knihovna je napsána zcela ve spravovaném kódu C#, což zajišťuje skutečnou multiplatformní kompatibilitu napříč Windows, Linux a macOS. Bez nativních binárních souborů nebo závislostí třetích stran se Aspose.Medical bezproblémově integruje do jakékoli aplikace .NET, od desktopových nástrojů po cloudové zdravotnické systémy.</p>

<p><strong>Klíčové funkce:</strong>
<ul>
    <li>Čtení a zápis souborů DICOM s plnou podporou všech standardních datových prvků DICOM a hodnotových reprezentací.</li>
    <li>Anonymizace souborů DICOM pro ochranu soukromí pacientů a zajištění souladu s HIPAA pomocí konfigurovatelných profilů důvěrnosti.</li>
    <li>Konverze obrazů DICOM do běžných formátů: JPEG, PNG, TIFF a zpět do DICOM z rastrových obrázků.</li>
    <li>Serializace a deserializace dat DICOM do JSON a XML pro integraci s moderními webovými službami a API.</li>
    <li>Překódování mezi transferovými syntaxemi včetně komprese JPEG, JPEG 2000, HTJ2K, JPEG-LS a RLE.</li>
    <li>Plná podpora víceframových obrazů DICOM a obrazových překryvů.</li>
    <li>Pokročilé zpracování obrazu s modalitními a VOI vyhledávacími tabulkami pro správné nastavení okna a úrovní.</li>
    <li>Podpora HTJ2K (High-Throughput JPEG 2000) pro kompresi lékařského zobrazování nové generace.</li>
    <li><strong>Již brzy:</strong> Konverze DICOM do NIfTI pro neuroimaging a výzkumné aplikace.</li>
</ul>
</p>

<p>Aspose.Medical for .NET je samostatná, soběstačná knihovna, která nevyžaduje instalaci žádných prohlížečů DICOM, softwaru pro lékařské zobrazování ani nativních kodeků. Je ideální pro zdravotnické IT systémy, integraci PACS, platformy telemedicíny a jakoukoli aplikaci vyžadující spolehlivé zpracování DICOM.</p>

     Čistá knihovna .NET - nulové závislosti

Na rozdíl od jiných knihoven pro lékařské zobrazování, které spoléhají na nativní kodeky nebo externí nástroje, je Aspose.Medical zcela soběstačný. Všechny obrazové kompresní kodeky (JPEG, JPEG 2000, HTJ2K, JPEG-LS, RLE) jsou implementovány v čistém .NET, což zajišťuje bezproblémové nasazení napříč Windows, Linux a macOS bez jakékoli platformově specifické konfigurace.     <h2 class="h2title">
     Anonymizace souborů DICOM pro soulad s ochranou soukromí
    </h2>
    <p>
     Ochrana soukromí pacientů je v zdravotnických aplikacích kritická. Aspose.Medical for .NET poskytuje komplexní možnosti anonymizace DICOM založené na profilech důvěrnosti DICOM PS 3.15. Můžete odstraňovat nebo upravovat identifikační informace pacienta (PII) při zachování klinické hodnoty zobrazovacích dat. Knihovna podporuje konfigurovatelné profily anonymizace, včetně možností zachování bezpečných soukromých atributů, UID, identifikátorů zařízení a institucí, charakteristik pacientů a longitudinálních časových informací.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Anonymizace souboru DICOM - C#
     </h3>
     <pre><code class="cs">// Načtení souboru DICOM
using var dicomFile = DicomFile.Open("patient_scan.dcm");

// Vytvoření anonymizéru s výchozím profilem
var profile = ConfidentialityProfile.CreateDefault();
var anonymizer = new Anonymizer(profile);

// Anonymizace a uložení
anonymizer.Anonymize(dicomFile, "anonymized_scan.dcm");</code></pre>
    </div>
   </div>
   <div class="col-lg-12">
    <h2 class="h2title">
     Konverze DICOM do obrazových formátů v .NET C#
    </h2>
    <p>
     Aspose.Medical for .NET usnadňuje konverzi obrazů DICOM do standardních obrazových formátů pro prohlížení, sdílení nebo integraci s nelékařskými systémy. Knihovna zpracovává veškerou složitost vykreslování obrazů DICOM, včetně správné aplikace modalitních LUT, VOI LUT (okno/úroveň) a fotometrické interpretace. Ať už potřebujete generovat náhledy, vytvářet obrazové exporty nebo budovat prohlížeč DICOM, API poskytuje vysoce kvalitní vykreslování s minimálním kódem.
    </p>
    <div class="codeblock" id="code">
     <h3>
      Konverze DICOM do PNG - C#
     </h3>
     <pre><code class="cs">// Načtení souboru DICOM
using var dicomFile = DicomFile.Open("chest_xray.dcm");

// Vykreslení do obrazu
var renderOptions = new GrayscaleRenderOptions
{
    WindowWidth = 400,
    WindowCenter = 40
};
var image = dicomFile.RenderImage(renderOptions);

// Uložení jako PNG
image.Save("chest_xray.png");</code></pre>

To je nezbytné pro integraci PACS, optimalizaci archivů a zajištění kompatibility napříč různými zdravotnickými systémy. Všechny kodeky jsou implementovány v čistém .NET, což zaručuje konzistentní chování napříč všemi platformami.

To umožňuje bezproblémovou integraci s RESTful API, systémy založenými na FHIR a webovými aplikacemi. Serializace je obousměrná - můžete také analyzovat JSON a XML zpět do datových sad DICOM.
